# [기획서] 동양미래대학교 노션 스타일 실시간 홈페이지

## 1. 프로젝트 목적
동양미래대학교(DMU)의 아이덴티티를 유지하면서, 세계적으로 인기 있는 협업 툴인 'Notion'의 깔끔한 디자인 시스템을 홈페이지로 구현합니다. 대학 생활 정보를 직관적으로 관리하고, 실시간 게시판을 통해 소통할 수 있는 프리미엄 개인 포털 구축을 목표로 합니다.

## 2. 디자인 시스템 (UI/UX)
- **테마**: Notion Dark Mode (배경색: #191919)
- **구성 요소**: 
  - **사이드바**: '최근 항목', '개인 페이지', '데이터베이스'로 구분된 노션 표준 레이아웃.
  - **대시보드**: 최근 방문 항목을 카드 형태로 보여주는 갤러리 뷰.
  - **Typography**: Inter 폰트를 사용하여 가독성을 극대화한 모던 스타일.
  - **Brand**: 동양미래대학교 공식 로고 및 캠퍼스 이미지를 활용한 브랜딩.

## 3. 기술 스택 (Tech Stack)
- **Frontend**: Vanilla HTML5, Tailwind CSS (CDN), Lucide Icons.
- **Backend**: Firebase 10 (Cloud Firestore) - 별도의 서버 구축 없이 실시간 데이터 연동.
- **Deployment**: 별도의 NPM 빌드 과정이 필요 없는 Standalone HTML 구조 (브라우저에서 즉시 실행 가능).

## 4. 메뉴 구성
- **Home**: 최근 방문 카드와 인사말이 포함된 전용 대시보드.
- **학교 소개**: 학교의 비전과 브랜딩 이미지 구성.
- **학과 소개**: 전공 안내 및 관련 정보.
- **나의 포트폴리오**: 개인 프로젝트 및 경력 관리 섹션.
- **관련 회사 & 학교 생활**: 취업 정보 및 캠퍼스 라이프 가이드.
- **실시간 게시판 (board.html)**: Firebase DB와 연동되어 누구나 글을 쓰고 실시간으로 읽을 수 있는 데이터베이스 뷰.

## 5. 핵심 프롬프트 요약 (재구현용)
> "동양미래대학교의 공식 브랜드 이미지와 노션(Notion)의 다크 모드 레이아웃을 결합한 홈페이지를 만들어줘. NPM을 사용하지 않고 HTML 파일 하나로 실행 가능해야 하며, Firebase CDN을 사용해 실시간 게시판 기능을 구현해줘. 메인 대시보드는 노션의 '최근 방문' 카드 스타일로 만들고, 게시판은 별도의 board.html로 분리해서 데이터베이스 테이블 형태로 제작해줘."
